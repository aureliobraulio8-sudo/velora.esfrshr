Dados e Privacidade
1. Recolha de Dados

Recolhemos apenas o e-mail e nome para autenticação. Não recolhemos dados sensíveis.

2. Armazenamento

As tuas conversas são guardadas localmente no teu dispositivo. Não armazenamos conversas nos nossos servidores.

3. Autenticação

O login é gerido pelo Firebase (Google), sujeito à Política de Privacidade da Google.

4. Partilha de Dados

Não vendemos, partilhamos nem cedemos os teus dados pessoais a terceiros, sem excepção.

5. Os Teus Direitos

Podes eliminar a tua conta e todos os dados a qualquer momento nas Definições.

6. Contacto

Para questões sobre privacidade: assistente.19@gmail.com
